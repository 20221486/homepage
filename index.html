HTML

Description: <html>

  • Hypertext Markup Language
  • The <html> element is the root element of an HTML document.
    It contains all other elements that make up the webpage.
  • It defines the beginning and end of the HTML document.

Description: <head>

  • The <head> element contains metadata and information about the
    document that isn't directly displayed on the webpage.
  • It provides essential details such as the document's title, character
    encoding, styles, and scripts.
  • Common Elements Inside <head>:
    • <title> Specifies the title of the webpage (displayed on the browser tab).
    • <meta> Provides metadata like character encoding, viewport settings, or
      description.
    • <link> Links external resources, such as CSS stylesheets.
    • <script> Includes JavaScript files or inline scripts.
    • <style> Contains internal CSS styles.

Description: <body>

  • The <body> element contains the content of the webpage that is visible
    to users.
  • It houses all the elements that make up the visual and interactive parts
    of the page, such as text, images, videos, forms, and links.
  • Common Elements Inside <body>: 
    • Headings       <h1> to <h6>
    • Paragraphs     <p>
